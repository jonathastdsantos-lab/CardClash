# Passo a Passo — Hospedar Política de Privacidade no GitHub Pages > Tempo total: ~15 minutos. Gratuito, sem cartão, URL profissional. --- ## O que você vai ter no final Uma URL pública tipo `https://[seuusuario].github.io/futcards-privacy/` com a política de privacidade carregando bonitinha. Essa URL você cola na Play Console. --- ## CAMINHO A — Pela interface web (recomendado, sem terminal) ### Passo 1 — Criar conta no GitHub (se ainda não tem) 1. Acesse https://github.com 2. Clique em **Sign up** 3. Use seu e-mail, crie uma senha, escolha um username (vai aparecer na URL — escolha algo profissional, ex.: `futcards` ou `seunome-dev`) 4. Confirme o e-mail > Esse username vai virar parte da URL pública. Escolha algo decente. ### Passo 2 — Criar o repositório 1. Logado no GitHub, clique no **+** no canto superior direito → **New repository** 2. **Repository name:** `futcards-privacy` (esse vai aparecer na URL) 3. **Description:** "Política de Privacidade do app Fut Cards" 4. Marque como **Public** (precisa ser público para o GitHub Pages funcionar grátis) 5. Marque **Add a README file** (vai criar o repositório já com um arquivo) 6. Clique em **Create repository** ### Passo 3 — Criar o arquivo da política 1. No repositório recém-criado, clique em **Add file** → **Create new file** 2. No nome do arquivo, digite: `index.html` 3. Cole o conteúdo HTML pronto abaixo no editor (próxima seção) 4. Role até o final da página, clique em **Commit changes** (botão verde) 5. Na janela que abrir, confirme clicando em **Commit changes** novamente ### Passo 4 — Ativar o GitHub Pages 1. No repositório, clique em **Settings** (engrenagem, no topo) 2. No menu lateral esquerdo, clique em **Pages** 3. Em **Source**, selecione **Deploy from a branch** 4. Em **Branch**, escolha **main** e **/ (root)** 5. Clique em **Save** 6. Aguarde 1-2 minutos. Recarregue a página. 7. Aparece em cima: `Your site is live at https://[seuusuario].github.io/futcards-privacy/` ✅ **Essa é a URL que você cola na Play Console.** ### Passo 5 — Testar Abre a URL no navegador. Se a página da política aparece bonitinha, deu certo. Se aparecer erro 404, espera mais 2-3 minutos (o deploy às vezes demora) e tenta de novo. --- ## HTML pronto para colar no `index.html` Copia tudo abaixo. Ajusta os campos `[entre colchetes]` antes de salvar. ```html Política de Privacidade — Fut Cards

Política de Privacidade

Fut Cards · Última atualização: [DATA]

A presente Política de Privacidade descreve como o aplicativo Fut Cards (doravante "App"), operado por [SEU NOME OU EMPRESA], CNPJ/CPF [NÚMERO], coleta, usa e protege as informações dos usuários.

1. Informações que coletamos

a) Informações fornecidas pelo usuário:

  • Nome e e-mail obtidos via login com Google, Facebook ou Apple;
  • Nome de usuário escolhido dentro do App;
  • Data de nascimento (verificação de idade).

b) Informações coletadas automaticamente:

  • Identificador do dispositivo (Device ID);
  • Modelo, sistema operacional e versão do dispositivo;
  • Dados de uso (telas visitadas, tempo de uso);
  • Endereço IP aproximado (segurança).

c) Conteúdo gerado:

  • Cartas colecionadas, trocas realizadas, resultados de partidas.

2. Como usamos as informações

  • Manter sua conta ativa e personalizada;
  • Permitir trocas e partidas entre usuários;
  • Melhorar a experiência do App;
  • Prevenir fraudes e violações dos termos;
  • Comunicar atualizações importantes;
  • Cumprir obrigações legais.

3. Compartilhamento de dados

Não vendemos seus dados. Compartilhamos apenas:

  • Com provedores de serviços (autenticação, armazenamento, análise) sob contrato;
  • Quando exigido por ordem judicial ou autoridade competente;
  • Em caso de fusão ou aquisição da empresa, com aviso prévio.

4. Seus direitos (LGPD)

Conforme a Lei 13.709/2018 (LGPD), você pode:

  • Confirmar a existência de tratamento de dados;
  • Acessar seus dados;
  • Corrigir dados incompletos ou desatualizados;
  • Solicitar a exclusão da sua conta e dados associados;
  • Revogar o consentimento.

Para exercer qualquer direito: [SEU E-MAIL]

5. Segurança

Aplicamos medidas técnicas e organizacionais razoáveis para proteger seus dados, incluindo criptografia em trânsito (HTTPS) e controles de acesso. Nenhum sistema é 100% seguro; comunicamos eventuais incidentes conforme a LGPD.

6. Crianças e adolescentes

O App é destinado a usuários a partir de 13 anos. Usuários entre 13 e 17 anos devem ter consentimento dos responsáveis. Não coletamos intencionalmente dados de menores de 13 anos.

7. Retenção

Mantemos seus dados enquanto a conta estiver ativa. Após a exclusão, os dados são apagados em até 30 dias, exceto quando a lei exigir retenção maior.

8. Alterações nesta política

Podemos atualizar esta Política. Mudanças relevantes serão comunicadas no App com aviso de pelo menos 30 dias.

9. Contato

Dúvidas sobre privacidade:
[SEU E-MAIL]
[SEU ENDEREÇO OPCIONAL]

Encarregado de Proteção de Dados (DPO):
[SEU NOME ou NOME DO RESPONSÁVEL]

© [ANO] Fut Cards. Todos os direitos reservados.
``` --- ## CAMINHO B — Pelo terminal (se você prefere git) ```bash # Cria a pasta local mkdir futcards-privacy && cd futcards-privacy # Cria o arquivo index.html (cola o HTML acima nele) nano index.html # ou code . para abrir no VS Code # Inicia git, conecta com seu repo git init git add index.html git commit -m "primeira versão da política" git branch -M main git remote add origin https://github.com/[seuusuario]/futcards-privacy.git git push -u origin main ``` Depois ativa GitHub Pages pelo painel do repositório (Settings → Pages → main branch), conforme Passo 4 acima. --- ## Quando precisar ATUALIZAR a política GitHub Pages é vivo — toda mudança que você fizer no `index.html` (via interface web ou git push) é refletida na URL em 1-2 minutos. Você não precisa de novo deploy nem nada. Boa prática quando atualizar: 1. Trocar a data no topo do documento ("Última atualização") 2. Avisar usuários no app com 30 dias de antecedência (LGPD exige) --- ## URL de exemplo Se seu username GitHub for `joaosilva` e o repositório for `futcards-privacy`, a URL final é: **https://joaosilva.github.io/futcards-privacy/** (Note a barra no final — algumas configurações de Pages exigem.) --- ## Solução de problemas comuns **"Página em branco"** → Provavelmente o nome do arquivo não é `index.html`. Esse nome é obrigatório. **"404 Not Found"** → GitHub Pages às vezes leva 5-10 minutos pra ficar online após ativar. Espera, recarrega. **"O ícone do cadeado quebrado"** → Acontece raramente. Em Settings → Pages, marque **Enforce HTTPS**. **"Quero trocar para um domínio próprio"** (ex.: `politica.futcards.com.br`) → Settings → Pages → Custom domain. Mas isso só faz sentido se você comprar o domínio. Por enquanto, a URL do GitHub Pages basta para a Play Console. --- ## Próximo passo Depois que a URL estiver no ar: 1. Abre o navegador anônimo, cola a URL, confirma que abre direito. 2. Volta na Play Console → **Política de Privacidade** → cola a URL → salva. 3. Marca a tarefa como concluída. Daí já dá pra atacar a próxima do guia (Detalhes do login).
